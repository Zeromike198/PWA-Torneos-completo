Manual de Usuario — PWA Torneos

Introducción
------------
Este manual explica al usuario y al administrador cómo usar la aplicación PWA Torneos.

1. Acceso a la aplicación
- Ir a la URL local (p.ej. http://localhost:5173)
- Desde Home puede ver resultados y torneos.
- Para tareas administrativas, haga clic en Login e ingrese sus credenciales.

2. Usuarios públicos
- Home: ver torneos recientes.
- Resultados: seleccionar torneo y filtrar por categoría o buscar atleta.

3. Administrador
- Dashboard: métricas y acceso rápido.
- Atletas: Registrar nuevo atleta -> completar nombre, club, género -> Guardar.
- Inscripciones: seleccionar torneo y atleta -> asignar categoría y dorsal -> Inscribir.
- Tiempos: seleccionar evento -> registrar tiempo mediante Start/Stop o ingreso manual.
- Resultados: ver lista ordenada por tiempo, exportar CSV (si disponible).

4. Recomendaciones
- Para registrar tiempos en vivo, use dos navegadores/ventanas: uno como cronometrista y otro como espectador.
- Mantenga copia de seguridad de la base de datos antes de importar SQL.
